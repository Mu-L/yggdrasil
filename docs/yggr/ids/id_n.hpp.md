## 主要内容

### 1. 定长ID模板类
#### `template<u64 LEN> class id_n`
- 实现了长度为 `LEN` 字节的定长ID类型，底层以字符串（`string_type`）存储。
- 支持拷贝、移动、赋值、交换、比较等操作。
- 提供与标准容器兼容的迭代器接口（`begin/end/rbegin/rend` 等）。
- 支持与其他容器类型的数据拷贝和赋值。
- 提供 `inner_buffer()` 方法访问底层数据。
- 支持与标准字符串类型的相互转换。

### 2. 依赖与特性
- 依赖 Yggdrasil 基础类型、算法、序列化、Boost 容器与算法等。
- 支持序列化、哈希、字节级比较、字节填充等高级功能。
- 适用于需要高性能、定长唯一标识的场景，如分布式系统、数据库主键、加密等。

### 3. 其他
- 文件包含大量模板元编程和类型萃取，保证类型安全和灵活性。
- 兼容C++11及以上标准，支持Boost库的相关特性。

## 用途说明
本文件为Yggdrasil系统提供了灵活且高效的多字节定长ID类型，适用于需要自定义长度ID、跨平台唯一标识、序列化传输等多种场景。
