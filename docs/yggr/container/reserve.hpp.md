# 文件：reserve.hpp

## 简介

`reserve.hpp` 是 yggdrasil 项目中用于为容器预分配空间（reserve）的头文件。该文件通过类型萃取和辅助类机制，为支持 `reserve` 操作的容器提供统一的预分配接口，提升代码的通用性和可维护性。

## 主要内容

- 头文件保护宏，防止重复包含。
- 依赖 reserve_helper.hpp，以及 Boost 的类型萃取和 SFINAE 工具。
- 提供 `reserve` 模板函数，对非 const 容器对象进行预分配操作。
- 内部通过 `container::detail::reserve_helper` 辅助类实现具体的预分配逻辑，自动适配不同容器类型。
- 导出 `reserve` 到yggdrasil命名空间，便于全局调用。

## 用法说明

- 适用于需要为容器预分配空间以提升性能的场景，支持标准容器、自定义容器等。
- 直接包含本头文件，并使用 `yggr::reserve(c, size)` 即可，无需关心容器类型细节。
- 仅对非常量容器对象有效，const 容器将被 SFINAE 排除。

## 设计说明

- 通过 SFINAE 和类型萃取机制，自动适配不同容器类型，提升代码通用性和安全性。
- 兼容 C++98/03/11/14/17 及 Boost 生态，适合跨平台和长期维护项目。
