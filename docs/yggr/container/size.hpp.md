# 文件：size.hpp

## 简介

`size.hpp` 是 yggdrasil 项目中用于统一获取容器大小的头文件。该文件基于 Boost.Range 的 `size`，并扩展了 `ssize`（带符号的 size）接口，方便在不同容器和算法中安全地获取元素数量。


## 主要内容

- 头文件保护宏，防止重复包含。
- 依赖 upper_types.hpp、range_size_ex.hpp 以及 Boost.Range。
- 直接导出 `boost::size` 到 `yggr::container::size`，用于获取容器元素数量。
- 提供 `ssize` 模板函数，返回带符号的容器大小，类型自动提升为更大的有符号整数，适合与算法和索引运算配合使用。
- 导出 `size` 和 `ssize` 到yggdrasil命名空间，便于全局调用。

## 用法说明

- 适用于需要统一获取各种容器大小的场景，支持标准容器、自定义容器和第三方容器。
- 直接包含本头文件，并使用 `yggr::size(cont)` 或 `yggr::ssize(cont)` 即可。

## 设计说明

- 通过类型萃取和类型提升机制，保证 `ssize` 返回类型的安全性和兼容性，防止溢出和类型不匹配问题。
- 兼容 C++98/03/11/14/17 及 Boost 生态，适合跨平台和长期维护项目。
