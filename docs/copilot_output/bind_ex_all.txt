# 文件：bind.hpp

该头文件属于 yggdrasil 项目，主要用于扩展 Boost.Bind 的功能，提供了 `bind_t` 对象的比较操作符重载。

## 主要内容

### 1. 头文件包含

```cpp
#include <yggr/bind/bind.hpp>
```
引入了 yggdrasil 项目下的 bind 相关实现。

### 2. 宏定义：_YGGR_PP_TMP_BIND_COMPARE

该宏定义了一组用于 `boost::_bi::bind_t` 类型对象的比较操作符重载：

- `operator==` 和 `operator!=` 针对相同模板参数的 `bind_t` 对象，调用其 `compare` 方法进行比较。
- 针对不同模板参数的 `bind_t` 对象，直接返回 `false`（==）或 `true`（!=）。

#### 作用

使得 `boost::_bi::bind_t` 对象可以直接使用 `==` 和 `!=` 进行比较，便于在容器、算法等场景下使用。

### 3. 命名空间

- 所有比较操作符重载均定义在 `boost::_bi` 命名空间内，确保与 Boost.Bind 的实现兼容。
- 预留了 `yggr::bind_ex` 命名空间，便于后续扩展。

### 4. 宏清理

```cpp
#undef _YGGR_PP_TMP_BIND_COMPARE
```
防止宏污染。

## 用法示例

```cpp
boost::_bi::bind_t<R, F, L> a = ...;
boost::_bi::bind_t<R, F, L> b = ...;
if (a == b) {
    // ...
}
```

## 注意事项

- 该文件依赖于 Boost.Bind 的内部实现，升级 Boost 版本时需注意兼容性。
- 仅对 `boost::_bi::bind_t` 类型对象有效。
